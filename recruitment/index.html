recruitment

我们是upwork上工作的特种兵，我们高效，我们搞笑。因为我们是线上工作，所以我们需要您再加入我们之前掌握基本的工作使用：git，github。另外由于我们会和国外的小伙伴合作，当然项目90%来自国外，如果您的英语能力比较强的话，那将会是您的一大优势。总之，只要您技术足够优秀，我们破格录取。期待您的加入。

工作模式

  1. 兼职
    • 时薪（5～40 dollars）(透漏下我的时薪目前刚到10dollars)
    • 固定金额
  2. 全职

招聘对象

  • 在职人员（希望您每周至少能抽出8个小时的时间来为我们工作）
  • 学生
    • 建议您关注下我们接下来两个月的博客教程。
    • 我们将会在您开学回来后，举办一次web技术大赛。
    • 奖金10000人民币(要加入我们哦)。
    • 如果您成功加入我们，雅思托福的考试费用我们就承包了。

招聘岗位

java web（3人）

任职要求 :

  1. 本科或以上学历，计算机软件或相关专业；
  2. 三年Java项目开发经验，具有独立开发和设计能力。Java基础扎实，熟悉互联网和web应用的相关开发技术；
  3. 熟悉面向对象和设计模式，深入理解Java开发框架，如Spring，mybatis等。熟悉开源框架，或应用服务器内部机制者优先 ；
  4. 熟悉数据库设计（Mysql），具有一定优化经验.

前端（3人）

任职要求 :

  1. 三年以上web前后端开发经验，一年以上移动终端开发经验；
  2. 熟练掌握html、css3，熟悉node.js、angularjs、jquery等css及其js框架；
  3. 熟悉移动端兼容性解决方案，能够解决多浏览器版本兼容问题；
  4. 熟练使用JS实现Json、XML格式的数据发送与数据解析；

UI/UX设计师（2人）

这个岗位我们只招UI/UX设计师，我只记得我们上一个雇用的设计师是45dollars／h，相当于月薪5万左右。
